AWS IoT Device Defender Agent SDK
==================================

Python SDK for AWS IoT Device Defender that provides metrics collection and reporting capabilities for IoT devices.

Features
--------

* Collect system and network metrics from IoT devices
* Support for both standalone and AWS IoT Greengrass deployments  
* Configurable metrics collection intervals
* JSON and CBOR report formats
* Custom metrics support
